# Hooks Reference

Custom hooks and key utility functions.

## useMarketData

```ts
function useMarketData(retryKey?: number): MarketDataState
```

Fetches prices, flex events, and household usage in parallel. Returns a
discriminated union:

- `{ status: 'loading' }`
- `{ status: 'ready', prices, flexEvents, householdUsage }`
- `{ status: 'error', error }`

Increment `retryKey` to re-fetch after an error.

**Data sources:**
- `/api/agile-prices` — live Octopus Agile tariff
- `/data/flexibility_opportunity.json` — flex event windows
- `/data/household_usage.csv` — three household profiles

**Source:** `src/hooks/useMarketData.ts`

## useTimeRange

```ts
function useTimeRange(items: { ts: number }[]): {
  fullRange: TimeRange;
  activeRange: TimeRange;
  isCustomRange: boolean;
  setRange: (range: TimeRange) => void;
  resetRange: () => void;
}
```

Manages a zoomable time window. `activeRange` equals `fullRange` until the
user zooms or pans. Invalid ranges (non-finite, zero-width) are silently
ignored. Ranges are inclusive on both ends — `fullRange.toTs` is the
timestamp of the last data point, not one past it.

**Source:** `src/hooks/useTimeRange.ts`

## usePriceStats

```ts
function usePriceStats(points: PricePoint[], range: TimeRange): PriceStats
```

```ts
interface PriceStats {
  min: { price: number; ts: number } | null;
  max: { price: number; ts: number } | null;
  total: number | null;
  count: number;
}
```

Uses binary search for efficient slicing — runs on every frame during drag
interactions. Range is inclusive on both ends: `[fromTs, toTs]`.

**Source:** `src/hooks/usePriceStats.ts`

## useUsageStats

```ts
function useUsageStats(
  usage: HouseholdUsageRow[],
  prices: PricePoint[],
  range: TimeRange,
  households: HouseholdKey | HouseholdKey[],
): UsageStats
```

```ts
interface UsageStats {
  totalKwh: number;
  estimatedCostPence: number;
  peak: { kwh: number; ts: number } | null;
  low: { kwh: number; ts: number } | null;
  count: number;
}
```

Accepts a single key or array of keys. Range is inclusive on both ends:
`[fromTs, toTs]`. Cost is estimated as `Σ(kWh × price)` per slot via
binary search price lookup.

**Source:** `src/hooks/useUsageStats.ts`

## findCheapestWindow

```ts
function findCheapestWindow(
  data: ReadonlyArray<{ ts: number; value: number }>,
  fullRange: TimeRange,
  durationMs: number,
): TimeRange | null
```

O(n) sliding window over sorted half-hourly data. Skips non-contiguous gaps.
Returns `null` if no valid window exists for the requested duration.

**Source:** `src/components/Features/MarketPrice/findCheapestWindow.ts`
