## Usage pattern

Icons are passed as React nodes, typically into `StatCard`'s `icon` prop or
rendered inline in component JSX:

```tsx
import { TbBoltFilled } from 'react-icons/tb';

<StatCard
  icon={<TbBoltFilled />}
  label="Current Price"
  value="24.7p"
  tone="accent"
/>
```

The `StatCard` icon slot renders at `--text-xs` (12px) in the card header,
coloured by the card's tone via `--_solid`.

**Source:** icon options defined in `src/components/UI/StatCard/iconOptions.tsx`
