## Glass panel

Used for bento tile cards and floating panels. Provides a translucent surface
with backdrop blur and saturation boost.

### Nested radius pattern

Cards use `--radius-inner` to create concentric arcs between the outer shell
and inner elements. The card ancestor computes:

```
--radius-inner: max(--radius-1, --radius-card − padding)
```

Standalone elements (tooltips, badges) that aren't inside a card fall back to
a sensible primitive value.

## Skeleton loading

A compositing-only pulse animation with zero layout/paint cost. A short
fade-in masks container-query layout shifts on first paint.

<div style={{ display: 'flex', flexDirection: 'column', gap: '0.5rem', margin: '0.5rem 0 1rem', padding: '1rem', background: 'var(--mono-subtle-bg)', borderRadius: '0.75rem' }}>
  <SkeletonBar />
  <SkeletonBar />
</div>

```scss
@include surface.skeleton-pulse;
```

<div style={{ margin: '0.25rem 0 1.5rem', lineHeight: 2 }}>
  <TokenChip token="--skeleton-fade-in" value="0.15s" />
  <TokenChip token="--skeleton-pulse" value="1.8s cycle" />
</div>

Respects `prefers-reduced-motion` — both animations are disabled when the
user has requested reduced motion.

## Z-index layers

Three layers, ordered from top to bottom:

## Page wash

The page background uses a layered aurora effect (edge vignette, accent-tinted
radial gradients, top sheen) applied via `surface.page-wash`. It renders as a
fixed `::before` pseudo-element behind all content.

```scss
@include surface.page-wash;
```
