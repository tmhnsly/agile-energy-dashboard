  </div>
);

# Typography

Two font families, an 8-step size scale, and 9 semantic presets.

**Source:** `src/styles/theme/typography.scss` (tokens), `src/styles/mixins/_typography.scss` (presets)

---

    Space Grotesk — Headings, numbers, labels
  </div>

</div>

<div style={{ margin: '0.5rem 0 1.5rem' }}>
  <div style={{ fontFamily: 'var(--font-body)', fontSize: 'var(--text-xl)', fontWeight: 400 }}>
    Inter — Body text, buttons, meta
  </div>
  <code style={{ fontSize: 'var(--text-xs)', color: 'var(--mono-text-low-contrast)' }}>--font-body</code>
</div>

Both are loaded via `next/font` and resolve through CSS custom properties with
system-font fallbacks.
